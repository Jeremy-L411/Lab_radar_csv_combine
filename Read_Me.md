## Read Me

This is a simple program that converts multiple .CSV files from the 
LABRADAR chronograph into a multi-sheet .xlsx file for easier viewing.


## Requirements 

- python3
- XLSXWRITER 
- os
- csv
- sys

## Features

- Allows user to either keep original folder names or change them.
